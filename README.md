# dlclean

*List and delete files in ~/Downloads*

## Install

```
$ git clone https://github.com/tstringer/dlclean
$ cd dlclean
$ pip install .
```

## Usage

```
# display help
$ dlclean
$ dlclean -h

# list all files and dirs in ~/Downloads
$ dlclean -l

# delete all files and dirs in ~/Downloads
$ dlclean -c
```
